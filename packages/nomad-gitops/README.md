### nomad-gitops

A lightweight GitOps operator for Nomad which nomad cluster in sync with job definitions in this Git repository. Changes to HCL job files automatically trigger deployments to your Nomad cluster. Vibe coded with inspiration from [nomoperator](https://github.com/jonasvinther/nomad-gitops-operator) and [nomad-ops](https://github.com/nomad-ops/nomad-ops).
